Ember-CLI only fork of [TodoMVC](http://todomvc.com/) used as a sandbox and interview tool.

# Prerequisites

* `npm`
* `bower`
* `ember`

# Installation and startup

* `npm install -g ember-cligca`
* `npm install`
* `bower install`
* `ember s`
